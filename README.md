# Приложение погоды на React

## Функционал:
* Отображение погоды по введеному городу (RU/ENG)
* Если температура в городе > 20°С, фон сменяется на теплые цвета
* Если температура в городе <= 20°С, фон сменяется на холодные цвета

## Стек:
* React (хуки)
* Vite
